# 3D_Human_Lung_DL_Segmentation
A deep learning 3D image segmentation pipeline tailored to segment human embryonic lungs imaged by lightsheet microscopy.

This repository contains the code and Jupyter notebooks used to segment 3D light sheet microscopy images of the developing human lung during the first trimester of gestation (weeks 5–14 after conception).

All classes and functions are documented in the source code. Example training, validation and evaluation data are also provided. The Jupyter notebooks provides thorough documentation and explains how to train, validate and use our custom deep learning image segmentation pipeline to generate 3D segmentation masks of the embryonic human lung for further downstream analysis.
